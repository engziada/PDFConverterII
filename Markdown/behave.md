# Unit: الوحدة الثالثة: مهارات حل المشكلات
## Slide: 34
### Lesson: أنواع المشكلات
### Content:
1. **مشكلات الترتيب:** إعادة تنظيم العناصر (مثل ترتيب حروف لكلمة).  
2. **مشكلات الاستقراء:** إيجاد علاقة بين عناصر (مثل تسلسل أرقام).

4. **مشكلات الإبصار:** حل يتطلب تفكيرًا عميقًا وإدراكًا مفاجئًا (Insight).
